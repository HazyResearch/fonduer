Packaging
=========

You can package the whole Fonduer pipeline (parsing, extraction, featurization, and classification) and deploy it to a remote place to serve.
To this end, we use `MLflow Model`_ as a storage format.

.. _MLflow Model: https://www.mlflow.org/docs/latest/models.html
